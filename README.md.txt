# WDN-ML-Optimization (Henrietta Water Distribution Network)

**Machine-learning–assisted modeling, surrogate-based optimization, and uncertainty analysis** for the Henrietta water distribution network (Texas), using Python, DOE/LHS, classical ML (RF/XGBoost/LightGBM), GNN surrogate, multi-objective optimization (NSGA-II), and robustness assessment.
